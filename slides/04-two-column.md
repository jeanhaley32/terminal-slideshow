# SLIDE 4: Two-Column Comparison Template

```
┌──────────────────────────────────────────────────────────────────────┐
│                            COMPARISON                                │
├─────────────────────────────────┬────────────────────────────────────┤
│           OPTION A              │            OPTION B                │
├─────────────────────────────────┼────────────────────────────────────┤
│                                 │                                    │
│  Pros:                          │  Pros:                             │
│  + First advantage              │  + First advantage                 │
│  + Second advantage             │  + Second advantage                │
│  + Third advantage              │  + Third advantage                 │
│                                 │                                    │
│  Cons:                          │  Cons:                             │
│  - First disadvantage           │  - First disadvantage              │
│  - Second disadvantage          │  - Second disadvantage             │
│                                 │                                    │
│  Best for:                      │  Best for:                         │
│  Scenario description           │  Scenario description              │
│                                 │                                    │
└─────────────────────────────────┴────────────────────────────────────┘
```

## Speaker Notes

Two-column layout for comparisons. Use `+` for pros and `-` for cons. Keep columns balanced in content length. The divider should be at approximately the midpoint.
