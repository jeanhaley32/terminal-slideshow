# SLIDE 6: Bullet List Template

```
┌──────────────────────────────────────────────────────────────────────┐
│                           KEY POINTS                                 │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  * First main point goes here with enough detail to be clear         │
│                                                                      │
│  * Second main point - keep each point to one or two lines           │
│    - Supporting detail or sub-point                                  │
│    - Another supporting detail                                       │
│                                                                      │
│  * Third main point explains another concept                         │
│    - Sub-point with specific example                                 │
│                                                                      │
│  * Fourth main point wraps up the list                               │
│                                                                      │
│  * Fifth point if needed (aim for 4-6 items max)                     │
│                                                                      │
│  ----------------------------------------------------------------    │
│  KEY TAKEAWAY: Summarize the most important point here.              │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

## Speaker Notes

Bullet list template for key points. Use `*` for main points and `-` for sub-points. Keep lists to 4-6 items. Add a key takeaway at the bottom if helpful.
